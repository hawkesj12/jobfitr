The journey · seven moments

From a blank prompt to an application, one focused card at a time

The chat fills the search by talking, the page lifts into results, and a single wide card holds the role in focus while the next-best waits, masked, below it. Each mockup is live — the gauges fill as you scroll.

The thread that runs through it

The assistant never fully leaves — and it's how you steer

The chat stays docked at the top as a control surface. It speaks on real events, not a chatty timer, and it's the one place you change what you're looking for — no "reset," no "retry." The left drawer dials the how; the chat changes the what.

◆ speaks on eventsEarned lines onlyA couple of passes in a row: "Nothing landing? Want to widen it?" A strong match: "This one scores high." Never filler.
◆ refine by talkingConfig is conversational"Actually, remote only." The model emits a partial update, it merges, the list re-scores. No controls to hunt for.
◆ clean by designTwo surfaces, two jobsChat changes the search; the left drawer narrows it. No retry button, no settings sprawl.
The line we hold

Honest engagement vs. dark pattern

Same mechanics the casinos and dating apps use — pointed at real value instead of against the user. The left column ships. The right column never does.

✓ Fair game — ships
  • One crafted peak — the page lift + the fit reveal, then a warm close.
  • Honest open loop — real roles waiting below, best-fit first.
  • Real return value — "14 fresh since Tuesday," true from the harvest.
  • Real scoreboard — the board counts actual applications.
  • Live, honest counts — filters show 34 → 12, never a fake number.
  • Reduced-motion honored — every moment degrades gracefully.
✕ Dark pattern — never
  • Confetti on everything — reward inflation; trains the eye to ignore it.
  • Manufactured scarcity — fake "3 people viewing this right now."
  • The near-miss — "you were 1 away." A hard red line.
  • Streak guilt — "don't lose your streak" notifications.
  • Synthetic precision — a "92%" that means nothing. (Cut in v2.)
  • Faked latency — typing dots as theater. Erodes trust.

The one-line test: does this animation or number carry information — a transition, a value, a real achievement — or is it decoration? If it's decoration, cut it.

Under the hood · why it's safe

Two planes, one gate

The AI talks and fills a config. The scoring engine reads a cached snapshot and never touches an external API. The only thing that crosses is a validated config — both the security boundary and the reason it stays fast and cheap.

AI plane · metered · streamed
chat input → /api/chat
→ OpenRouter (cheap model)
→ emits set_config( )
turn cap · per-IP limit · daily ceiling
→only a
config dict

Scoring plane · free · zero-network
/api/score → config_from_dict
→ score the cached jobs.json
→ ranked JSON + rank / tier
never calls an external API ✓

Proven by spike: hostile AI output (DROP TABLE, <script>, rm -rf /, hallucinated keys) collapses into a harmless search config — unknown keys ignored, bad types defaulted, strings treated only as keywords. The attack ceiling is "a weird search."

Where the decisions stand

One resolved, one open, one to bless

Resolved in review ✓

The gauge shows no synthetic %

Dropping the percentage retires the "what does 100% mean" question entirely. The fill carries strength; the corner carries a real word and a real position.

Remaining sub-choice: tier word ("Strong fit") as the label — my pick, it answers "should I apply" — with rank ("#1 of 34") as context. Or flip them. Both are shown above.

Decision · open

Two filter surfaces, or one?

v2 splits them: quick discrete filters in the header toolbar (sort, remote, tags, source); precise/ranged ones in the left drawer (salary, fit floor, exclude).

Recommendation: keep the split — common filters stay one tap away, the drawer holds the rest. If it feels like two homes, we consolidate into the drawer only.

Decision · bless the direction

Professional, with life — not a game

The whole board deliberately rejects the slot-machine kit (tumbler, near-miss, streak-guilt, confetti-on-everything) for one crafted peak and honest loops. That restraint is what makes it read as a real product and it's the stronger portfolio story.

Confirm this as the north star and the next step is folding it into BUILD-PLAN.md as Phase E.
